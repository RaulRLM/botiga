# Botiga

## Descripció del projecte
Aquest projecte consisteix en la creació d'una botiga en línia bàsica utilitzant HTML, CSS i JavaScript. L’objectiu és desenvolupar un catàleg de productes, afegir filtres i cerca, implementar un carret de compra amb persistència i un flux de checkout amb validació de formulari.

## Tecnologies
- HTML5
- CSS3
- JavaScript (vanilla JS)
- LocalStorage per a la persistència del carret
- JSON per a les dades dels productes

## Objectius del projecte
1. **Catàleg de productes:** Llistat de productes amb dades mock (JSON), visualitzat amb cards responsives.
2. **Filtres i cerca:** Permetre filtrar els productes per categoria i cercar per nom.
3. **Carret de compra:** Afegir, eliminar i modificar productes amb persistència entre recàrregues.
4. **Checkout:** Modal o drawer amb formulari validat, resum de compra i missatge d’èxit.

## Gestió del projecte
Aquest projecte utilitza un **GitHub Project (Kanban)** per organitzar les tasques.  
Les 5 issues creades són:
- **Issue 1:** Maquetació base i estructura del projecte
- **Issue 2:** Llistat de productes (catàleg) amb dades JSON
- **Issue 3:** Filtre per categoria + cerca per text
- **Issue 4:** Carret de compra amb persistència (LocalStorage)
- **Issue 5:** Modal/Drawer de Checkout + validació de formulari

Les issues estan etiquetades amb labels de tipus i prioritat, i assignades a les columnes adequades del Kanban:
- **Backlog:** Issue 3 i Issue 5  
- **To do:** Issue 1 i Issue 2  
- **In progress:** Issue 4
